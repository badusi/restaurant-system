# Restaurant System

A comprehensive web-based management application designed to streamline daily restaurant operations. This system automates the process of taking customer orders, managing menu items, tracking table availability, and handling billing, while providing an administrative dashboard to monitor sales and performance.

## 🚀 Features
* **Digital Menu Management:** Dynamic CRUD operations for dishes, categories, pricing, and availability.
* **Order & Billing Tracking:** Smooth order placement for waitstaff or customers, with automated receipt generation.
* **Table Reservation:** Real-time tracking of vacant, occupied, and reserved restaurant tables.
* **Admin Analytics Dashboard:** Comprehensive overview of daily sales, popular dishes, and overall revenue reports.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL (via XAMPP phpMyAdmin)

## 💻 How to Run the Project
1. Download or clone this repository as a `.zip` file.
2. Extract the folder and move it into your XAMPP installation directory: `C:\xampp\htdocs\`
3. Open the **XAMPP Control Panel** and start both **Apache** and **MySQL**.
4. Open your browser and go to `http://localhost/phpmyadmin/` to create a new database.
5. Import your project's `.sql` database file (if included) into phpMyAdmin.
6. Open a new tab in your browser and visit: `http://localhost/restaurant-system/`
